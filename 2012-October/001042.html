Hello list,

Last time I made many changes in code of X2Go Client.
New Features:

-Support for SSH Proxy (HTTP-Proxy was developed by <Heinrich
Schuchardt> xypron.glpk at gmx.de)
-Support for SSH broker
-Some improvements in common broker code
-better support for encrypted SSH-Key
-improvements in SSH connection classes
-hide system tray icon before closing client
etc.

I have also wrote some documentation which can help you by developing
your own X2Go broker. Mike will edit it and publish on X2Go Wiki soon.

You can get a source code from our GIT:
http://code.x2go.org/gitweb

the windows binary:
http://code.x2go.org/releases/binary-win32/x2goclient/x2goclient-3.99.3.0-pre3-setup.exe

the windows binary with obsolete PulseAudio server:
http://code.x2go.org/releases/binary-win32/x2goclient/with-old-pulseaudio-0.9.6/x2goclient-3.99.3.0.interims-pre3-setup.exe

Debian packages will be built soon.
