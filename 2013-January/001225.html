This is because we added a file to x2goserver called  
/etc/x2go/x2goagent.options.

In this file we disabled the GLX and XFIXES extension in x2goagent  
(which is the XServer for the X2Go session.

Re-enable GLX if needed. XFIXES causes problems with kubuntu and other  
applications, if you never had problems, re-enable that, as well.

Re-enable := comment out the corresponding line in  
/etc/x2go/x2goagent.options.
