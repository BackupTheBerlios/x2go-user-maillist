Then you have to places where to add applications.

  1. /etc/x2go/applications/* (publish system-wide)
  2. $HOME/.x2go/applications/* (publish per-user)

STEPS:

   o install the software you want to publish via dpkg/rpm
   o find the startmenu/desktop icon in /usr/share/applications/
   o copy your preferred .desktop file to one of the above locations
