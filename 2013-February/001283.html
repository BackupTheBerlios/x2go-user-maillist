The wiki page desperately needs an update.

The x2goplugin is available as .deb package for Debian and Ubuntu.

On your webserver you can install x2goplugin-provider. The package  
contains a little example that shows how to configure a webpage, so  
that the x2goplugin gets launched once the webpage is visited.

The x2goplugin package has to be installed on the systems from where  
the browser connects to the x2goplugin-provider site.
