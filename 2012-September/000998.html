I suppose you are in GNOME 2 on Ubuntu 10.04. So there are two ways...

1.
Use the GNOME keyboard settings tool to configured the client-side keyboard.
This way is to go, if you use the X2Go server machine locally _and_ via X2Go.

2.
Let x2gosetkeyboard do the keyboard customization job.

   a. Remove all keyboard settings for your user under GNOME(2)
   b. set keyboard type to ,,auto'' in the X2Go session profile and
      leave the layout empty

Once logged in, please report what ,,setxkbmap -print'' spits out to stdout...
