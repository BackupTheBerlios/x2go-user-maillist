After you have installed the latest X2go server-side package the  
command ,,x2goversion'' should output something like this:

mike at minobo:~$ x2goversion
cups-x2go: 3.0.0.2
x2goagent: 3.5.0.2
            ^^^^^^^
x2gognomebindings: 2.0.1.2
x2golxdebindings: 1.0.1.0
x2goserver: 3.0.99.5
x2goserver-extensions: 3.0.99.5

Especially the x2goagent version should be as pointed out above.
