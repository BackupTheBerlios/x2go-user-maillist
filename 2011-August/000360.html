I did some more thinking/tinkering and think that what Brian says makes more sense

>Not sure if you remember but when I was trying to get PyHoca working on a
>chrooted Ubuntu environment on the
>nook color tablet I was also getting the RandR error in the logs.
>
>Initial authentication would work but before I'd see any remote display it
>would crash.
>

>running Ubuntu's x-server at the time.
