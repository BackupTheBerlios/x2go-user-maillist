I had seen some problems similar to this in the past when the user account
used a shell other than bash.   In the Ubuntu I was using, the default
shell was /bin/sh, which failed.   I also saw tickets regarding issues with
tcsh.

So, I would confirm you're using /bin/bash and test again.
