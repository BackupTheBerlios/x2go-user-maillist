Please switch to loglevel=debug in /etc/x2go/x2goserver.conf and see  
what syslog brings then.

If you have not updated x2goserver.conf yet then place this stanza  
into the file

[log]
# possible levels are: emerg, alert, crit, err, warning, notice, info, debug
loglevel=debug
