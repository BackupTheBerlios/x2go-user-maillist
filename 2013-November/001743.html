Just a wild guess: check if there is a group named "x2godesktopsharing"
in /etc/group file (or its alternative in OpenSuse).

If not, try to create it (man groupadd). It may fail due to length of
gid name being limited in some *nix systems (e.g. Arch maxes at 16
characters). If that is the case, you - or the package maintainer -
would have to patch the source code with a shorter gid name.
