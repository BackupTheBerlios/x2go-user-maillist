Brent.

In my experiments with Unity & Gnome I ended up just installing a different
Desktop Environment.
Mate, Xfce or LXDE are all easy to install and all work well with X2GO.

If you are using ubuntu
xfce -> $ sudo apt-get install xubuntu-desktop
lxde -> $ sudo apt-get install lubuntu-desktop
mate which is gnome2 based -> follow this:
http://www.ubuntu-news.net/2013/11/13/install-mate-desktop-1-6-with-mintmenu-in-ubuntu-12-04-13-10/

I would assume these work the same on debian
