Hi Kermit,

On  Di 19 Nov 2013 21:36:26 CET, Short, Kermit wrote:

> Greetings!  I am used to installing the x2gognomebindings package on a
> system when I install the x2goserver.
>
> Today I tried and aptitude returned with package not found.
>
> Sure enough, I aptitude search and the package is no longer present.
> What happened, and is there a work around?

The package is only available in the lucid ppa of X2Go. In later  
versions (starting with natty) GNOMEv2 was removed and replaced by  
GNOMEv3 and x2gognomebindings is for GNOMEv2 only.

On git.x2go.org [1] we provide a package called x2gomatebindings.git  
which is the continuation of x2gognomebindings, only adapted to MATE.

The package is currently only built for our Debian archive, as it  
requires some MATE packages as build-deps (which was not trivial with  
Launchpad the last time I tried). Build it from source if needed.

Once MATE has entered Debian (and with it: Ubuntu), I will provide  
x2gomatebindings via our PPA on Launchpad as binary package.

Mike

[1] http://code.x2go.org/gitweb?p=x2gomatebindings.git;a=summary
