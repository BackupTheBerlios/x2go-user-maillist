If you configure network-manager system-wide and from a config file  
(i.e. you do not use the nm-applet utility or something similar  
GUI'ish)...

[ -> Google for this by yourself, there is some Ubuntu-Wiki help around... ]

... then you can do this:

   o install a minimal Debian squeeze from a Debian CD
   o install x2gothinclient from our repos

... and then start using an X2Go thin client that is installed on the  
netbooks hard disk.
