any errors/critical/warning in event log.   I'm getting the following 
errors from Event Log:

sshd: PID 10416: error: Could not load host key: /etc/ssh_host_key
sshd: PID 10416: error: Could not load host key: /etc/ssh_host_rsa_key
sshd: PID 10472: error: Couldn't create pid file "/var/run/sshd.pid": No
