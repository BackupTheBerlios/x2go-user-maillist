Hi, all.  I haven't really been following this thread but, in my
experience, black screens in Ubuntu were usually caused by using dash
instead of bash as a shell.  I don't know if the current version of
x2goserver has specified #!/bin/bash instead of #!/bin/sh.  If not, try
changing them or change the default shell in Ubuntu with
rm /bin/sh
ln -s /bin/bash /bin/sh
