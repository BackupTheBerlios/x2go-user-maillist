> It sounds like the key on the known_hosts file has changed.  I do not
> recall where that is stored on the Windows client but, if you can find
> that known_hosts file in the X2Go directories and delete it, I think you
> will be fine - John

It's either in %USERPROFILE%\.x2go/ssh/known_hosts or  
%USERPROFILE%\.ssh\known_hosts.

Try to search your C:\Users folder (or C:\Documents and Settings on  
WinXP) for a file of the name known_hosts. Remove that one, as John  
said and you should be fine.
