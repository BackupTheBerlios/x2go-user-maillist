Heinz is currently working on this in his spare time (during the day  
he is busy with earning money ;-) ). So, yes, it is in process. And:  
Heinz probably is the best person to give information on the plugin  
status.

What we want to provide after Heinz has made some progress is a new  
x2goclient package called ,,x2goplugin-provider''. This package (for  
Debian/Ubuntu) will auto-configure an apache server so that after  
installation who can click on a http://localhost/x2goplugin link (or  
similar) and the X2Go plugin with some default session is launched  
within your browser. As I said, this is also work in progress.
