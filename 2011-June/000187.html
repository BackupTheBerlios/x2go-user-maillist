I wanted to try Gnome3 with Gnome Shell and couldn't make it work on
Debian, so I installed Fedora 15.

packages for Mandriva. Getting them to install on Fedora wasn't that
difficult, I've documented it here:
http://wiki.x2go.org/installing_x2goclient_fedora

Please note that there is no support - from me or anyone else - on
this. That said, it worked pretty much out of the box.
