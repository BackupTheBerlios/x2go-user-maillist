Thanks, team.  Delighted to have the new version. For any who have been
used to installing vcxsrv separately, it looks like it is the embedded X
server now rather than Xming so a separate installation is not
necessary.  vcxsrv released 1.10.3.1 just this past Monday so we may
want to update that for 4.0.  So far, I've only tested with the included
vcxsrv.

I've been heavily testing the Windows client from XP for several hours
now and there are a number of issues - some Windows specific, some not.
I'm running against two different X2Go servers - one running 3.0.1-5 and
the other 3.99  I'll enumerate them here and then spawn a separate email
for each one.

1) Very serious issues with image packing
2) Alt tab not working as expected in full screen
3) Multiple X2goclient instances fail
4) General sluggishness
5) Task bar icon does not disappear on close
6) Miscellaneous - not necessarily X2Go
7) Clean up of old installations
8) Return from full screen mode not working
