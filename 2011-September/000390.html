nxproxy. So I don't expect an strace of X2goclient (or Pyhoca-GUI) will
give information.

But more: I think it's something on the server what crashes. Because
when only something on the client would crash, the session would not be
lost. When I open a new session, there is no old session. So when I
think well, it must be something on the server what crashes.
