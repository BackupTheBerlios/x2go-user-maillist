On Tuesday 20 September 2011 17:58:13 Paul van der Vlis wrote:
> Hello,
> 
> To use Pyhoca-GUI you need 4 steps:
> 
> - start Pyhoca-GUI, what only adds an icon
> - find and open the icon
> - connect to a server
> - start a session to that server
> 
> I think for many normal people this is "complex".

Afaik pyhoca-gui is meant for admins that connect to several servers during 
one work-session.

For normal users there are x2goclient and x2goclient-cli, which do:

> - start the program, so that it's visable and not only an icon
> - start a connection and session to a server

You can even give these parameters to start a specific session so they only 
have to enter their username and password.

Have fun,
