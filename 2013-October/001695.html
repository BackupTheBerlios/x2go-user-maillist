with the present features of X2Go, providing a MS Windows support  
infrastructure like you know from team viewer is not possible.

X2Go can provide a helpdesk-like support workflow for machines running  
Linux desktops. For this, you have to install X2Go server and X2Go  
Desktop Sharing on those machines, so the machines have to be under  
your administrative control.
