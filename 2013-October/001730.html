Hi,

I am using X2go broker with SSH proxy to have X2go servers sit behind
the broker not directly reachable from the internet. Like this, I need
Users have access to ports 443 (usually not a problem) and SSH (very
often a problem). Problem is that users tend to sit behind transparent
http/https proxies, so SSH won't get out. I would like to see
everything being done through HTTPs so we can trick proxies.

Is this going to be implemented in the near future or maybe even
already possible somehow?
