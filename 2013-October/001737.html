> 4)  When I try:

>
> I get an error window:
> Login Failed!  Please try again

The default setup should provide the broker under this URL:
x2goclient --broker-url=http://xxx.xxx.xxx.xxx/x2gobroker/plain/inifile

If you change something in /etc/x2go/apache*.conf make sure to restart  
the Apache webserver.

> When I try:
>
> x2goclient --broker-url="ssh://xxx.xxx.xxx.xxx/usr/bin/x2gobroker  
> --backend=inifile"
>
> I get:
> sh: 1: /usr/bin/x2gobroker: not found
>
> FYI:  on the server, there isn't an x2gobroker in /usr/bin....there  
> is an x2gobroker script in /usr/sbin.
>
> If I try:
> x2goclient --broker-url="ssh://xxx.xxx.xxx.xxx/usr/sbin/x2gobroker  
> --backend=inifile"
>

For SSH broker support you need X2Go Session Broker (package:  
x2gobroker) from our nightly builds(!!!). The upstream version should  
be 0.0.3.0.
