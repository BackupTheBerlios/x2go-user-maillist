Hi Arton,

please notice, that official language of x2go mailing lists is English.

Now back to your problem. X2Go Client saving his settings in Windows
registry. Actually, you should be always able to restore original
settings using configuration dialog. But if you experiencing problems
with it, you can delete X2Go Client settings from Windows registry.

- run regedit.exe
- change to HKEY_CURRENT_USER/Software/Obviously Nice/x2goclient
- remove entry "settings"

This should reset X-Server settings to it's defaults.

Regards,
Alex
