Hi Andreas,

On Di 25 Jun 2013 13:09:36 CEST Andreas Yankopolus wrote:

> I'm trying to figure out how to use X2Go to connect with a machine  
> at a private IP address via a public ssh server. I'm currently using  
> ssh authentication forwarding to get to that machine:
>
> ssh -A -t andreas at public.ip.address ssh -A andreas at private.ip.address

SSH agent forwardings are only supported by the Python X2Go Client  
named PyHoca-GUI. X2Go Client does not support forwarding of agent  
auth requests.
