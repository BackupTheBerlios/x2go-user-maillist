Hi Andreas,

On Di 25 Jun 2013 16:29:41 CEST Andreas Yankopolus wrote:

> Mike,
>
>> SSH agent forwardings are only supported by the Python X2Go Client  
>> named PyHoca-GUI. X2Go Client does not support forwarding of agent  
>> auth requests.
>
> It's not looking like PyHoca-GUI plays well with OS X. Is it  
> possible to set up port forwarding manually and point X2Go at  
> localhost when connecting? Or maybe there's some other option?

Aftern thinking it through, you should be able to use the SSH proxy  
feature (Connection tab of X2Go Client). Configure the  
public.ip.address as the SSH proxy and the private.ip.address as X2Go  
Host. Enable SSH Agent authentication for both target machines (SSH  
Proxy, X2Go Host).
