There are two approaches.

1.
Every member of the posix group x2godesktopsharing may share each  
other's desktop.

2.
Normally, each Posix user on a UniX machine has a primary group of the  
same name (e.g. user foo has primary group foo, user bar has primary  
group bar).

To give user bar access to user foo's desktop, add user bar to the group foo.
