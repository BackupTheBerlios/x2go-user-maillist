The x2godesktopsharing utility is for sharing desktops across different users.

If do the same test as this: http://bit.ly/TdpcxE with two different  
user accounts, this should not work, right? (shivvvveeeer....)

If you then launch x2godesktopsharing on the first desktop (the one  
you want to share), then you can access this desktop via  
x2godesktopsharing via another user account.

Adding users to x2godesktopsharing happens once another user connects  
to your account. The GUI, of course, can be improved here: adding  
users before they connect via the GUI could be a possible feature  
request.
