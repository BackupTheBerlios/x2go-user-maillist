Is the server-side user member of the fuse group?

If yes, set debug loglevel in /etc/x2go/x2goserver.conf and watch  
/var/log/syslog.

If that does not help: what X2Go version do you have on the server  
(run x2goversion command). On what distro are you, what version of  
that distro do you use (all server-side).
