Hi

Running x2go-server on Kubuntu 13.04 is problematic since it has the new (KDE) "intelligent" screen manager. It remembers the screen resolution and automatically sets it (It's nice for physical screens). This causes big problems when you start a session the first time(where I got an overly large resolution) and when going from a large to a small screen.
It even remember the screen resolution between sessions - so no help logging in/out.

The workaround for now is to remove the kscreen package - but in the longer run it needs to be fixed.
I've filed a bug report in kscreen - but I don't know what will happen to it.
https://bugs.kde.org/show_bug.cgi?id=319065
